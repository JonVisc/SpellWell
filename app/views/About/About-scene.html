How To Play
When you start the game it will list out 4 choices of a word, only one of the choices is truly correct.

When you select a choice the game will indicate whether or not it is the correct choice. If you make all of the correct choices in the level of 5 words, you progress. If you guess incorrectly the game will start the current level over.

A constant timer is ticking down your score but don't worry if the score reaches zero the game doesn't end! If it is a particularly difficult level you can spend time really learning the words.
But try to be quick to get the best score!
